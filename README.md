# 📖 Quran Web App

A modern and responsive Quran web application built with Next.js.  
Users can explore all 114 Surahs, read Arabic verses with English translations, search ayahs, and customize reading settings.

## ✨ Features

- 📜 Display all **114 Surahs** (Arabic & English names)
- 📖 View **Ayahs with Arabic text and English translation**
- 🔍 **Search functionality** (by translation text)
- ⚙️ **Settings Panel**
  - Change Arabic font (multiple options)
  - Adjust Arabic font size
  - Adjust translation font size
  - Persist settings using localStorage
- 📱 Fully responsive design (mobile + desktop)

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (SSG)
- **Backend:** Node.js
- **Styling:** Tailwind CSS
- **API:** https://api.alquran.cloud/v1/surah

## ⚙️ Setup & Run Locally

```bash
# Clone the repository
git clone https://github.com/xunaiet-faruk/Quran-web

# Go to project folder
cd Quran-web

# Install dependencies
npm install

# Run development server
npm run dev
